You have access to the following built-in predicates:

### Basic

* `none?`
* `eql?`
* `key?`

### Types

* `type?`
* `str?`
* `int?`
* `float?`
* `decimal?`
* `bool?`
* `date?`
* `date_time?`
* `time?`
* `array?`
* `hash?`

### Number, String, Collection

* `empty?`
* `filled?`
* `gt?`
* `gteq?`
* `lt?`
* `lteq?`
* `max_size?`
* `min_size?`
* `size?(int)`
* `size?(range)`
* `format?`
* `inclusion?`
* `exclusion?`
